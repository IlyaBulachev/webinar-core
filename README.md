# Ядро системы «Онлайн система семинаров» 
 
Дипломная работа по специальности 09.02.07 «Информационные системы и программирование». 
 
## Технологии 
 
- **Сервер:** Node.js 20, TypeScript, Express, Prisma ORM 
- **Медиасервер:** Janus Gateway (SFU) 
- **База данных:** PostgreSQL 15 
- **Кеш/брокер:** Redis 
- **Клиент:** React 18, TypeScript 
 
## Быстрый запуск 
 
1. Установите Docker и Docker Compose 
2. Склонируйте репозиторий 
3. Скопируйте .env.example в .env и заполните значения 
4. Запустите: docker-compose up -d 
5. Откройте в браузере: http://localhost
